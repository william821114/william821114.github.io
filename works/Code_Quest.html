Code Quest—勇者啊，用猜數字拯救世界吧！

Code Quest是一款結合猜數字（又稱終極密碼，Bulls and Cows）與角色培育的益智RPG。

遊戲規格

  • 開發團隊：NCU 計算機實習 一上小組
  • 團隊成員：柯政佑、蔣彥亭、謝昆荃、邱義傑、林子寗
  • 開發語言：JAVA
  • 遊戲素材：RPG Maker 2000—Alex、The King of Fighters '96—瑪瓊亞(マチュア；Mature)、Unlight地圖標的、純手繪動畫（by 柯政佑&林子寗）

遊戲介紹


在Code Quest的世界中，玩家將扮演勇者—大中天，透過猜數字的神奇力量，從魔王—黑暗女巫的手中拯救眾生。

遊戲世界涵蓋了古木參天的森林、天寒地凍的冰原、燋金爍石的火山，以及陰森恐怖的女巫古堡。玩家將從名為START的起始點一路向右，展開一連串的冒險。
遊戲提供多條攻略路線，要以擊敗黑暗女巫為首要目標，還是要挑戰所有攻略點，一切交由玩家自己選擇。選擇攻略地點後，便會觸發事件。除了遭遇怪物的戰鬥事件外，也有機會觸發可以獲得稀有道具的寶箱事件。

Code Quest 戰鬥畫面

Code Quest 戰鬥畫面

戰鬥觸發後，會進入戰鬥畫面。玩家可以在畫面左下、右下方查看自己與敵人的生命值（Life）及攻擊力（ATK），並在畫面下方中間的輸入區輸入猜測的數字來發動攻擊。
系統會記錄玩家每次猜測的數字，並提供「幾A幾B」的情報，供玩家做下一步的猜測。

【猜數字遊戲 規則說明】轉自維基百科：猜數字
　　每猜一組數字，出數者就要根據這個數字給出幾A幾B，其中A前面的數
表示位置正確的數的個數，而B前的數字表示數字正確而位置不對的數的
個數。
　　如正確答案為5234，而猜的人猜5346，則是1A2B，其中有一個5的位置
對了，記為1A，而3和4這兩個數字對了，而位置沒對，因此記為2B，合起
來就是1A2B。
　　接著猜的人再根據出題者的幾A幾B繼續猜，直到猜中為止。

至於戰鬥傷害的部分，則是根據每次猜測數字的正確程度而定。猜中一個位置正確的數字（即1A），可以給予怪物一倍攻擊力的傷害，猜中兩個數字，則給予兩倍攻擊力傷害，依此類推。相反地，不論是沒猜中位置的數字（即B），還是連數字本身都沒猜對，怪物都會給予玩家相應的傷害。
另外，若玩家成功猜出所有數字，則可以對怪物產生一擊必殺的效果，直接取得戰鬥的勝利。


戰鬥結束後，玩家有機會獲得怪物掉落的道具。不論是戰鬥勝利從怪物身上取得的道具，或是在寶箱事件中取得的道具，都能在地圖畫面中點選「ITEM」來查看及使用。

遊戲中提供三個儲存格，玩家可以隨時在地圖畫面中點選「SAVE/LOAD」儲存遊戲進度。

時下有許多以簡單的益智小遊戲結合RPG概念的作品，像是結合記憶翻牌的マグナメモリア、結合轉珠的龍族拼圖等等。Code Quest嘗試採用具思考性的猜數字與RPG做結合，期望這前所未有的組合能讓玩家有全新的遊戲體驗。
